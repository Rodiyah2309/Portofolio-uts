Tentang Saya

Hai, saya Rodiyah, mahasiswa Sistem Informasi di STMIK Dharmapala Riau! Saya memiliki kenangan manis dengan matematika di sekolah, karena matematika itu menyenangkan (tetapi tidak dengan soal soal yang sulit, hehe!). Dulu, saya suka bermain game dan menonton anime karena pandemi membuat saya lebih banyak menghabiskan waktu di rumah. Tapi, setelah kuliah dan bergabung dengan BEM, saya menemukan kesenangan baru dalam berinteraksi dengan orang lain dan membuat perbedaan. Saya senang bertemu dengan orang baru dan memiliki pengalaman baru.

Sebelum masuk kuliah, saya pernah mengikuti les tentang Microsoft Word, Microsoft Excel, dan PowerPoint, dan saya berhasil mendapatkan nilai yang bagus. Pengalaman ini membuat saya lebih percaya diri dalam menggunakan teknologi dan mempersiapkan saya untuk menghadapi tantangan di bidang Sistem Informasi. Saya berharap dapat menggunakan kemampuan saya untuk membuat perbedaan di dunia dan mencapai kesuksesan yang saya impikan.
